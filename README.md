# UmbraSync

<p align="center">
  <img src="https://repo.umbra-sync.net/images/logo.png" alt="UmbraSync"/>
</p>


Projet francophone porté par un Rôliste pour les Rôlistes. Basé sur les travaux de DarkArchon et du projet Mare Synchronos.

## Fonctionnalités principales
- **AutoDetect** : regroupe la détection de proximité (publication/lecture pour envoyer des invitations aux joueurs à portée), l’annuaire SyncFinder public (inscription planifiée, durée d’exposition, statut temps réel) et le flux d’invitations rapides.
- **Bulle d'écriture** (*Inspiré de [RTyping](https://github.com/apetih/rtyping)*) : indicateur de saisie léger qui apparaît sur les nameplates ou directement dans le monde (même quand les plaques sont masquées) et fonctionne autant avec le chat natif qu’avec ChatTwo.
- **Nouvelle interface** : UI compacte et revisitée (Chara Data Hub, panneaux analytiques, notifications regroupées) optimisée pour Dalamud, avec navigation par onglets.
- **Partage MCDF** : Partagez le MCDF de vos personnages avec vos paires directement dans Umbra.
